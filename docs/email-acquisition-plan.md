# OutreachHub - 海外客户邮箱获取方案

## 一、方案概述

本方案为国内出海企业提供一套完整的、可落地执行的海外客户邮箱获取解决方案，涵盖从数据采集到邮箱验证的全流程。

### 2. 智能邮箱推测

基于已知信息推测邮箱：

```
已知条件：
- 公司域名：example.com
- 姓名：John Smith

常见邮箱格式：
- john.smith@example.com
- jsmith@example.com
- john@example.com
- smith.john@example.com
- john_smith@example.com

验证流程：
1. 生成所有可能的邮箱组合
2. 通过MX记录验证域名有效性
3. 通过SMTP验证邮箱是否存在
4. 标记验证结果
```

## 三、邮箱验证流程

### 验证层级

```
Level 1: 格式验证（正则表达式）
    ↓ 通过
Level 2: 域名验证（MX记录查询）
    ↓ 通过
Level 3: SMTP验证（邮箱是否存在）
    ↓ 通过
Level 4: 第三方验证（MillionVerifier/ZeroBounce）
    ↓ 通过
Level 5: 发送测试邮件验证
```

### 验证API对比

| 服务 | 价格 | 准确率 | 速度 |
|------|------|--------|------|
| **MillionVerifier** | $0.0005/封 | 98%+ | 快 |
| **ZeroBounce** | $0.007/封 | 99%+ | 中 |
| **Hunter** | 免费50次/月 | 95%+ | 快 |
| **EmailListVerify** | $0.0004/封 | 97%+ | 快 |

## 四、数据清洗与去重

### 清洗规则

1. **格式标准化**：统一小写，去除空格
2. **域名过滤**：排除免费邮箱（gmail, yahoo, hotmail）
3. **角色邮箱排除**：排除 info@, sales@, support@, contact@
4. **去重策略**：基于邮箱地址+公司域名组合去重
5. **黑名单过滤**：排除已知的无效域名和投诉邮箱

### 数据质量评分

```
评分维度（满分100）：
- 邮箱验证状态：40分
- 职位匹配度：20分
- 公司规模：15分
- 行业相关性：15分
- 数据新鲜度：10分

质量等级：
- A级（80-100）：优先发送
- B级（60-79）：正常发送
- C级（40-59）：谨慎发送
- D级（0-39）：不建议发送
```

## 五、合规性要求

### 1. GDPR合规（欧洲）

- 必须有合法的数据处理依据
- 提供数据删除机制
- 邮件必须包含退订链接
- 不能发送未经同意的营销邮件（B2B有例外）

### 2. CAN-SPAM合规（美国）

- 必须包含真实发件人信息
- 必须提供有效的退订机制
- 7天内处理退订请求
- 主题行不能误导

### 3. 中国数据出境

- 遵守《个人信息保护法》
- 数据存储在境内服务器
- 出境数据需安全评估
